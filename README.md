# CV Webseite - Harry Schleiss

## Projektbeschreibung
Dieses Projekt wurde im Modul Web Programming Basics erstellt.

Die Webseite dient als persöhnliche CV-Webseite und wurde mit HTML, CSS und Javascript entwickelt.
- Darkmode
- Login-Seite mit Passwort
- Reaction Test (kleines Game)
- Responsive Design für Mobile und Desktop

## Verwendete Technologien
- HTML
- CSS
- JavaScript

## Funktionen
- Navigation zu verschiedenen Bereichen
- Responsive Layout
- Darkmode
- Passwort-Login
- Reaction Test

## Projektstruktur
- "index.html" → Login-Seite
- "cv-harry-schleiss-2026-05-29.html" → Hauptseite der CV-Webseite
- "style.css" → Styling
- "script.js" → Javascript Funktionen
- "images/" → Bilder und Hintergründe
